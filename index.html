Methodology & Data Sources

Data Collection Architecture

The dataset driving this dashboard is automatically aggregated using a custom Python backend. The system executes a daily paginated scrape across Google News RSS, ensuring complete historical coverage.

  • Source Engine: Google News Global Feeds (English)
  • Frequency: Daily Automated CRON Job via GitHub Actions
  • Search Query: "Hormuz"

NLP & Deep Content Analysis

Advanced Natural Language Processing (NLP) is applied to every collected article:

  • Sentiment & Framing: Powered by TextBlob polarity scoring and custom fear/diplomacy lexicons.
  • Entity Extraction (NER): Driven by spaCy (en_core_web_sm) to identify actors (GPE) and organizations (ORG).
  • Topic Modeling & Network: Unsupervised K-Means clustering on TF-IDF vectors, and Co-occurrence edge generation for social network graphs.
